# Usage
<!-- usage -->
```sh-session
$ npm install -g llabtools
$ llabtools COMMAND
running command...
$ llabtools (-v|--version|version)
llabtools/0.0.1 darwin-x64 node-v11.9.0
$ llabtools --help [COMMAND]
USAGE
  $ llabtools COMMAND
...
```
